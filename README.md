# EverPeak Analysis – Sprint

🎯 Objetivo: Documentar tu análisis

Este repositorio contiene el análisis realizado durante el sprint del caso **EverPeak**.  
El caso EverPeak simula un escenario de retail con datos que incluyen valores faltantes, outliers y problemas de calidad, diseñados para reproducir situaciones reales de negocio.

---

## 📘 Descripción del caso EverPeak
El dataset `everpeak_dataset` incluye órdenes de clientes con distintos problemas de calidad de datos.  
El objetivo del análisis es limpiar, explorar y generar insights de negocio a partir de estos datos.

---

## 📂 Qué contiene el notebook
- Limpieza de datos  
- Exploratory Data Analysis (EDA)  
- Distribuciones y detección de outliers  
- Construcción de funnels y cohortes  
- Simulaciones de escenarios  
- Conclusiones del sprint  

Archivo principal:  
- `notebooks/everpeak_analysis.ipynb`
